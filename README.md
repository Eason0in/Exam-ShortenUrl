# 縮網址

此專案提供使用者將冗長的網址縮短

### 安裝

1.開啟終端機(Terminal)cd 到存放專案本機位置並執行:

```
git clone https://github.com/Eason0in/Exam-ShortenUrl.git
```

2.初始

```
cd Exam-ShortenUrl  //切至專案資料夾
```

```
npm install  //安裝套件
```

3.開啟程式

```
npm run start  //執行程式
```

終端顯示 `mongodb connected!` 即啟動完成，請至[http://localhost:3000](http://localhost:3000)開始使用程式
